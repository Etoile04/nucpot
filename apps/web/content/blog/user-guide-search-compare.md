# 高级搜索与材料对比功能指南

在掌握了基础的浏览操作后，本指南将介绍 NucPot 数据库的高级功能，帮助您更高效地获取所需的核材料物性数据。

## 高级搜索

### 搜索语法

NucPot 支持结构化搜索语法，允许您精确指定搜索条件：

| 语法 | 示例 | 说明 |
|------|------|------|
| 关键词搜索 | `锆合金` | 全文搜索匹配 |
| 属性筛选 | `热导率:>10` | 搜索热导率大于 10 的数据 |
| 范围查询 | `温度:300-1200` | 搜索 300–1200K 温度范围 |
| 组合搜索 | `Zr 热导率:>10` | 多条件组合查询 |
| 材料限定 | `material:UO2` | 在特定材料内搜索 |

### 搜索结果排序

搜索结果支持多种排序方式：

1. **相关度排序**：默认按与搜索条件的匹配程度排序
2. **温度排序**：按数据记录的温度值排列
3. **时间排序**：按数据录入或更新的时间排列
4. **来源排序**：按数据来源的权威性排序

### 搜索技巧

- 使用**引号**精确匹配：`"Zircaloy-4"` 仅匹配完整短语
- 使用**排除符**过滤不相关结果：`锆合金 -腐蚀` 排除腐蚀相关结果
- 使用**通配符**模糊搜索：`Zr*` 匹配所有以 Zr 开头的条目

## 材料对比功能

### 创建对比

材料对比工具允许您同时查看多种材料的物性数据，便于横向分析：

1. 在材料列表中勾选需要对比的材料（最多 4 种）
2. 点击「对比」按钮进入对比视图
3. 选择需要对比的物性参数

### 对比视图

对比视图以表格形式呈现，每列代表一种材料，每行为一个物性参数：

| 物性参数 | Zircaloy-2 | Zircaloy-4 | M5 | ZIRLO™ |
|----------|------------|------------|-----|--------|
| 密度 (g/cm³) | 6.51 | 6.55 | 6.51 | 6.52 |
| 熔点 (°C) | 1855 | 1855 | 1760 | 1850 |
| 热导率 (W/m·K) | 16.0 | 15.9 | 16.0 | 16.0 |
| 热膨胀系数 (μm/m·K) | 5.9 | 5.7 | 5.9 | 6.0 |

> **注意**：对比数据均标注了测试条件（温度、环境等），请确认对比条件一致后再做结论。

### 导出对比结果

对比结果支持以下导出格式：

- **CSV**：适用于 Excel 等电子表格软件
- **JSON**：适用于程序化处理和 API 集成
- **PDF 报告**：适用于存档和正式文档

## 实用场景

### 场景一：新型包壳材料选型

当需要为新型反应堆选择包壳材料时：

1. 使用高级搜索筛选所有包壳材料
2. 设置目标温度范围和力学性能阈值
3. 利用对比功能横向比较候选材料
4. 导出对比报告用于评审

### 场景二：文献数据验证

当需要验证实验数据与已有文献的一致性时：

1. 按材料名称和测试条件精确搜索
2. 对比数据库记录与实验结果
3. 查看数据来源和不确定性评估
