# 浏览材料数据库：快速入门指南

欢迎使用 NucPot 核燃料与材料物性数据库。本指南将帮助您快速上手浏览和查询核材料数据。

## 数据库概览

NucPot 数据库收录了核燃料循环各环节中关键材料的物性数据，包括但不限于：

- **燃料材料**：UO₂、UN、MOX 燃料等
- **包壳材料**：锆合金（Zircaloy-2、Zircaloy-4、M5）、不锈钢等
- **冷却剂材料**：轻水、重水、液态金属等
- **慢化材料**：石墨、铍等

## 使用筛选器

数据库提供了多维度的筛选功能，帮助您快速定位所需材料：

1. **按材料类别筛选**：在左侧导航栏选择材料类别（燃料、包壳、冷却剂等）
2. **按化学成分筛选**：输入元素符号或化学式进行精确搜索
3. **按温度范围筛选**：设置温度区间以获取特定温度范围内的物性数据
4. **按数据来源筛选**：选择数据来源（实验测量、文献综述、模型计算等）

## 导航本体树

本体树是 NucPot 的核心组织结构，按照核材料科学的知识体系进行分类：

```
核材料
├── 燃料材料
│   ├── 二氧化铀 (UO₂)
│   ├── 氮化铀 (UN)
│   └── 碳化铀 (UC)
├── 包壳材料
│   ├── 锆合金
│   └── 不锈钢
└── 结构材料
    ├── 石墨
    └── 铍
```

点击本体树中的节点可展开子类别，点击叶节点可直接查看该材料的物性数据列表。

## 查看材料详情

每个材料页面包含以下信息：

| 数据项 | 说明 |
|--------|------|
| 基本信息 | 化学式、晶体结构、密度、熔点 |
| 热物性 | 热导率、比热容、热膨胀系数 |
| 力学性能 | 弹性模量、屈服强度、断裂韧性 |
| 辐照效应 | 蠕变速率、辐照硬化、肿胀行为 |
| 数据来源 | 实验条件、参考文献、不确定性说明 |
